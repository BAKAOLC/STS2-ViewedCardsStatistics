# STS2-ViewedCardsStatistics

A Slay the Spire 2 mod that tracks and displays statistics for cards, relics, potions, and monsters you've encountered in the game.

## Features

- Track seen and picked counts for cards, relics, and potions
- Track monster encounters
- Per-character statistics breakdown
- Display stats on hover tips in the compendium
- Import data from existing run history
- Multi-language support (English, Simplified Chinese, Japanese, Korean)
- Console commands (`vcs reimport`, `vcs verbose`, `vcs clear`)

## Installation

Place the mod files in your Slay the Spire 2 mods directory.
